Code QR dynamique — usage en salle de classe

Page ouverte en fichier local Les requêtes vers l’API ne fonctionnent pas depuis file://. Indiquez l’URL du serveur ci-dessous (ex. http://127.0.0.1:8765) ou ouvrez cette page depuis le navigateur à cette adresse.

Démarrer un appel

Aucune session active. Saisissez la salle ou le cours, puis affichez le QR au vidéoprojecteur. Les élèves scannent avec l’application (Wi‑Fi établissement requis en production).

Pour le scan mobile → Google : dans .env, définissez PUBLIC_BASE_URL=http://VOTRE_IP:8765 (commande ipconfig), redémarrez le serveur, et autorisez le port au pare-feu. Les élèves doivent être sur le même Wi‑Fi.
— —

Les élèves scannent ce code avec leur téléphone : ils sont redirigés vers la connexion Google, puis la présence est enregistrée. Le code se renouvelle automatiquement.
